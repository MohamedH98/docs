## Setup your development

<Warning>You will first need to have node and npm/yarn installed</Warning>

### Step 1. Unzip your downloaded folder:

- Mac - double click to unzip your file
- Windows - extract zip files

### Step 2. Open the unzipped template in your code editor:

VSCode documentation can be found [here](https://learn.microsoft.com/en-us/visualstudio/ide/develop-code-in-visual-studio-without-projects-or-solutions?view=vs-2022)

### Step 3. Install your dependencies

Open up your terminal in your code editor and install your dependencies.

<CodeGroup>

```bash npm
npm install
```

```bash yarn
yarn install
```

</CodeGroup>

### Step 4. Install the sanity cli
<CodeGroup>

```bash npm
# Running the CLI without global installation
npx -y sanity@3 [command]


# Alternatively installing the CLI globally
npm install --global sanity@3
```

```bash yarn
# Running the CLI without global installation
npx -y sanity@3 [command]


# Alternatively installing the CLI globally
yarn global add sanity@3
pnpm install --global sanity@3
```

</CodeGroup>

### Step 5. Create a sanity project

```bash
npx create sanity@3 init --env
```
