<Warning>You will first need to have a github account</Warning>

### Step 1. Create a new github repository:

Navigate to this [link](https://github.com/new) and create a new github repository, call it whatever you like (set your repository to private)

### Step 2. Push your local respository to the remote branch:

You should see the following if the deploy successfully went through:

```bash
git remote add origin <REMOTE_URL>
git branch -M main
git push -u origin main
```
